# ChatMemOllama

一个 基于 aivfs的项目
ChatMenOllama.py位于AI虚拟文件系统根目录
/bin/ 负责连接至微信公众号

aivfs是一个开发中的项目，正在本地快速迭代开发
详情请查看本人仓库

此外，还将构建一个类似于GNU core utilies 的基础项目

更新日志/计划将发布在微信公众号（LIghtJUNction）
正式版具体发布时间未定
